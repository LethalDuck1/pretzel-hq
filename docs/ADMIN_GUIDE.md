# Pretzel HQ — Admin + Player Guide

This build focuses on stability: no loading/splash screens, and admin login stays compatible even if env variable names change.

## Required Netlify environment variables
- NETLIFY_SITE_ID
- NETLIFY_AUTH_TOKEN
- ADMIN_PASSWORD (or ADMIN_PASS / ADMIN_PASSCODE)
- ADMIN_USER (optional; defaults to admin)

## Troubleshooting
If admin login says not configured, ensure ADMIN_PASSWORD is set in the Netlify site environment variables and redeploy.

If you renamed env vars, this build accepts:
- ADMIN_PASSWORD, ADMIN_PASS, ADMIN_PASSCODE, LEGACY_ADMIN_PASSWORD, PRETZEL_ADMIN_PASSWORD
- ADMIN_USER, LEGACY_ADMIN_USER, PRETZEL_ADMIN_USER

## Security note
Do not hardcode passwords in the repo. Use Netlify env vars.

## Push notifications
This app uses Web Push. iPhone requires Add to Home Screen.

## Data storage
Game state and message history are stored in Netlify Blobs.
